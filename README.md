# swift-standards.org

Placeholder for the future [swift-standards.org](https://swift-standards.org) website — the public home for the Standards layer (Layer 2) of the Swift Institute ecosystem. The Standards layer answers *"what is specified externally?"* — specification implementations (RFCs, ISO standards, IETF, W3C, etc.).

Website planned for future. Current ecosystem content lives at https://swift-institute.org.
